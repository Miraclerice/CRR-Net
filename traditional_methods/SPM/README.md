## SPM
Batch script for SPM12 old Normalise.

### Requirements
The code has been successfully tested using SPM12 in MATLAB R2024a.


### Usage
- Download SPM12 from [github repo](https://github.com/spm/spm12/releases) and add the extracted directory to the MATLAB path according to the (official installation instructions)[https://www.fil.ion.ucl.ac.uk/spm/docs/installation/matlab/].

- Please modify the dataset path (base_dir in the script) before running the script.
